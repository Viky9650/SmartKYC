document_service.py — Step 2 · Primary Path
Gemini Vision API
Image + schema-aware prompt → structured JSON · fields + per-field confidence + MRZ lines + quality
Truncated JSON → partial recovery parser (regex salvage of fields block)
Vision returned fields?
YES — Merge MRZ
Step 2a · Supplement
MRZ Parser (ICAO TD3) on Vision's MRZ lines

surname · given names · DOB · nationality · expiry · doc number · sex
NO — OCR Fallback
Step 3 · Fallback
Tesseract OCR

Step 4
MRZ Parser on OCR text
ICAO TD3 line pair detection

Regex Parsers

Step 6 · _build_result()
Sanitise · Assemble full_name · Score confidence

Phase 7 — Risk Aggregation
RiskAggregationAgent
Weighted Final Score
final_score = Σ(agent_score × weight) ÷ Σ(weights of run agents)
Sanctions 30% · PEP 25% · Identity 15% · Registry 15% · Adverse Media 10% · Transactions 5%
risk_level: LOW <40 · MEDIUM 40–59 · HIGH 60–79 · CRITICAL ≥80
All flags deduplicated · written to Case.risk_score + Case.risk_level in DB
Case status → review
